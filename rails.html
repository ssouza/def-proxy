JSpec Rails Integration

No additional gems are required for JSpec to integrate with Ruby on Rails. Simply initialize a project using the command below at the root of your Rails project: 

$ jspec init --rails

JSpec will create a 'jspec' directory, and ... your done! The rest of JSpec's functionality is exactly the same, get started by binding JSpec using the command below and edit 'public/javascripts/application.js' (or any other JS) and JSpec will run your specifications. 

$ jspec

Alternatively just as you may with regular JSpec projects, run your specs using Rhino: 

jspec run --rhino

Or bind with Rhino 

jspec --rhino
