def-proxy

This is def-proxy, a small set of classes to facilitate easily working with dynamic proxies. The library aims to stay out of your way as much as possible, offering the following features.

Features

  • Simple mappings between proxy interface and back end delegate.
  • A clean, lightweight builder DSL for configuring mappings.
  • Support for declarative mappings using annotations.

Planned Features

  • We'll start with a stable API I think. :)

Cheat Sheet

Creating a new proxy.... 

final ProxyFactory factory = new AnnotationDrivenProxyFactory();
final Map delegate = new HashMap();
delegate.put("bar-code", "ABC55C666F");
delegate.put("product-id", "12345");
	
final Item proxy = factory.createProxy(delegate, Item.class);
assert (proxy.barcode() == "ABC55C666F");
assert (proxy.productId() == 12345);

Configuring your proxy interface 

@ProxyInterface(delegate = StoredItem.class)
public interface Item {
	@ProxyDelegated(methodName = "get")
	@ProxyArguments(value = {"bar-code"}, direction = Prefix)
	String barcode();

	@ProxyDelegated(methodName = "get")
	@ProxyArguments(value = {"product-id"}, direction = Prefix)
	@ProxyTypeConverter(provider = IntOfStringConverter.class)
	int productId();
}
